# ASSESS-DHT Evidence Generation Support Tool

Interactive tool to help digital health technology developers identify applicable HTA evidence recommendations across Europe.

## 🚀 Live Tool
**[Launch the Tool →](https://[taylorant-commit].github.io/assess-evidence-tool/)**

## 📊 Features
- **161 evidence recommendations** across 9 HTA domains
- **Smart filtering** by technology characteristics
- **PICO framework** generation (optional)
- **Domain selection** to focus on specific areas
- **PDF export** functionality

## 🏛️ About
Part of the ASSESS-DHT project, funded by the European Union's Horizon Europe Programme under Grant Agreement Number 101137347.

**Website:** [assess-dht.eu](https://assess-dht.eu)
